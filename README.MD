# Telegram-бот "Кто хочет стать миллионером?"

Телеграмм игра аналог кто хочет стать миллионером.

## Как запустить бота

1. **Установите зависимости:**
   Убедитесь, что у вас установлен Python (версии 3.8 и выше). Выполните команду для установки нужных библиотек:
   ```bash
   pip install -r requirements.txt
   ```

2. **Настройте переменные окружения:**
   - Скопируйте файл `.env.example` в `.env` (если он еще не создан):
     ```bash
     cp .env.example .env
     ```
   - Откройте файл `.env` и замените `your_telegram_bot_token_here` на токен вашего бота, полученный у [BotFather](https://t.me/BotFather) в Telegram.

3. **Запустите бота:**
   ```bash
   python main.py
   ```
